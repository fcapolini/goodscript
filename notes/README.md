# GoodScript Development Notes

This directory contains implementation details, planning documents, and technical guides that document the development process of the GoodScript compiler.

## Directory Structure

- **`implementation/`** - Technical implementation details and completion reports
- **`planning/`** - Phase planning documents and feature roadmaps
- **`guides/`** - User-facing guides for specific features

## Main Documentation

For primary documentation, see:
- `/README.md` - Project overview
- `/compiler/README.md` - Compiler package documentation
- `/compiler/docs/` - Language specification and architecture
- `/examples/` - Example programs demonstrating features

## Contributing

When documenting new features or implementation details, please place files in the appropriate subdirectory:
- Implementation details → `implementation/`
- Planning/design docs → `planning/`
- User guides → `guides/`
